Intelligent Scissor using MTALAB & C++

Based on Eric N. Mortensen, William A. Barrett Paper, Source Code , Email: xliuba@connect.ust.hk, shuangaj@connect.ust.hk


User Interface

  • Task Bar
    Open: open and load source image from computer disk.
    Save Contour: save current image with contour marked.
    Save Mask: save compositing mask for PhotoShop.
    Zoom In: zoom in.
    Zoom Out: zoom out.
    Pan: move zoomed image displayed in the window.

  • Mouse Operation
    Ctrl + Left Click: select first seed.
    Left Click: select following seed.
    Enter: finish the current contour.
    Back Space: when scissoring, delete the last seed; otherwise, delete selected contour.

  • Work Mode
    Image&Contour: display current contour and seeds on the source image lively.
    Image Only: display the source image.
    Gradient Map: display the gradient of source image.

  • Debug Mode
    Pixel Node: a 3W*3H graph with original image pixel colors at the center of each 3*3 window and black elsewhere.
    Cost Graph: a 3W*3H graph with original image pixel colors at the center of each 3*3 window and 8 linkcosts surrounding it.
    Path Tree: a 3W*3H graph with each 3*3 window displaying the minimum linkcost direction starting from center.
    Min Path: a minimum cost path from seed to current cursor position is displayed lively on original image.

  • Mouse Pos: display current cursor position.

  • RGB: display current cursor position's RGB value.

Implementation

  1. Image Processing:
    i. Generate cost graph (3W*3H) from source image.
    ii. Construct minimum path tree with Dijkstra's algorithm.

  2. Data Structure:
    i. Design our own interface for MATLAB to calculate each pixel's cost using C++ containter.
    ii. Save each pixel's parent index with 2D array in memory.
